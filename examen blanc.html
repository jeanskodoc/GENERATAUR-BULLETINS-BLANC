Sauvegarde et Restauration Complète

Exportez ou importez toutes les données de l'application en un seul fichier

Instructions d'Import/Export

Export :

  • Cliquez sur "Exporter" pour télécharger un fichier JSON
  • Le fichier contient les données au format structuré
  • Vous pouvez modifier le fichier JSON avec un éditeur de texte
  • Conservez une copie de sauvegarde régulièrement

Import :

  • Sélectionnez "Remplacer" pour écraser les données existantes
  • Sélectionnez "Ajouter" pour fusionner avec les données existantes
  • Assurez-vous que le fichier JSON a le bon format
  • Les IDs en double seront automatiquement ajustés

Format JSON attendu :

{
  "students": [
    {"id": 1, "lastName": "Dupont", "firstName": "Jean", "matricule": "2023001", "exam": "BEPC"}
  ],
  "subjects": [
    {"id": 1, "name": "Mathématiques", "coefficient": 4, "exam": "BEPC"}
  ],
  "grades": [
    {"id": 1, "studentId": 1, "subjectId": 1, "value": 15.5, "teacher": "M. Martin"}
  ]
}
